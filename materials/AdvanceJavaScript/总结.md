### 流行趋势

1. PWA(Progressive Web Apps，渐进式web 应用)运用现代的 web API以及传统的渐进式增强策略来创建跨平台web应用程序。

2. 低代码

- 少量的代码，通过拖拽和配置的方式实现应用的开发。

3. 微前端：多个前端应用整合为一个应用。每个前端应用单独开发，独立部署，应用间松耦合，应用内高耦合。

- single-spa，主流微前端框架;
- qiankun：基于single-spa二次开发，目前最完善最流行的框架。

4. web 3D
